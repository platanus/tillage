# Ruby module

Installs rbenv version manager and some plugins.
It will also install the ruby version defined in http://ruby.platan.us/latest

## Dependencies: `Brewfile`

- rbenv
- rbenv-vars
- rbenv-aliases
- ruby-build

## Gems: `gems`

Installs the following gems globally for the latest ruby:

- rails
- negroku
- rubocop
- potassium
